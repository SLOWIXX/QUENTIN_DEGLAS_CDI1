Exercice 2 : Identifier les rôles dans une application web

site : https://www.leonard-de-vinci.net/
Outils : extension Wappalyzer ( https://chromewebstore.google.com/detail/wappalyzer-technology-pro/gppongmhjkpfnbhagpmjfkannfbllamg?hl=fr  )

Front : 

Languages : HTML / CSS / JavaScript

Technologies : 

jQuery / jQuery UI : simplifie les interactions JavaScript (par exemple les animations).

DataTables : pour afficher des tableaux interactifs.

Select2 : améliore l’apparence et les fonctionnalités des menus déroulants.

SweetAlert2 : pour afficher des fenêtres de dialogue stylisées.

Dropzone : permet le glisser-déposer de fichiers pour les uploads.

Moment.js : gère les dates et heures.

Font Awesome : Fournit des icônes.

CodeMirror / Ace : éditeurs de code intégrés dans la page.

Raphael / Highcharts / amCharts : librairies pour afficher des graphiques.

HSTS



Back : 

PHP : langage côté serveur qui gère la logique métier, les connexions aux bases de données, l’affichage dynamique, etc.

SimpleSAMLphp : utilisé pour l’authentification via le protocole SAML (connexion sécurisée via une identité externe).

Cloudflare : protège le site contre les attaques, améliore la vitesse.

cdnjs : héberge des ressources JS et CSS utilisées par le frontend, pour optimiser les performances.
